# Larareact

This is larareact repo for session 3 and 4.

## Installation

install laravel and react

```bash
composer create-project laravel/laravel larareact
```

```bash
composer require laravel/ui

```

```bash
php artisan ui react
```

```bash
npm install
```

```bash
npm run dev
```

```bash
php artisan serve
```
